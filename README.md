## Instação

<br>

### Clone o repositório

```
git clone
```

### Acesse o diretorio

```
cd <nome-da-pasta>
```

### Instale as dependências

```
npm install
```

### Inicie a aplicação

```
npm run start
```

### A aplicação, por padrão, fica na porta:

```
http://localhost:3000/
```

### Testes da aplicação

```
npm run test
```
